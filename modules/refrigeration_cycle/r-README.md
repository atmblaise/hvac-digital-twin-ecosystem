# Refrigeration Cycle Simulator

## Overview
This module simulates vapor compression refrigeration cycles used in HVAC systems.

## Key Components
- Compressor
- Condenser
- Expansion valve
- Evaporator

## Key Concepts
- COP (Coefficient of Performance)
- Energy balance across components
- Phase change thermodynamics

## Applications
- Air conditioners
- Refrigerators
- Industrial cooling systems
