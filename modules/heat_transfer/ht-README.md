# Heat Transfer Module

## Overview
This module simulates heat transfer processes in HVAC and industrial systems including conduction, convection, and radiation.

## Key Concepts
- Fourier’s Law
- Newton’s Law of Cooling
- Overall heat transfer coefficient

## Core Equations
- $ Q = hA(T_s - T_∞) $
- $ Q = kA(ΔT/L) $

## Applications
- Heat exchangers
- Building envelopes
- Cooling coils
- Insulation analysis
