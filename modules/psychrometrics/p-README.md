# Psychrometrics Engine

## Overview
This module models moist air properties and air-conditioning processes used in HVAC systems.

## Key Concepts
- Humidity ratio
- Relative humidity
- Dry bulb and wet bulb temperatures
- Enthalpy of moist air

## Applications
- Air conditioning systems
- Cooling and dehumidification
- Ventilation analysis
